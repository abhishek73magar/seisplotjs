parseGeoJSON

src/usgsgeojson.ts

Parses geojson from USGS feeds. Not all fields are parsed, just basic origin and magnitude along with the id. See https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php

parseGeoJSON(geojson: USGSGeoJsonSummary): EventParameters
Parameters
geojson (USGSGeoJsonSummary) text from USGS feed
Returns
EventParameters: EventParameters, which holds an array of Quake objects
