Seisplotjs Gallery

Very small examples of web components from seisplotjs.

  • <sp-channel-code-input>

    A set of text input boxes for network, station, location and channel codes;

  • <sp-channel-list>

    A selectable list of channels.

  • <sp-channel-search>

    Build an FDSN Station query for channels.

  • <sp-channel-table>

    A table of information for a list of channels.

  • <sp-earthquake-search>

    Build an FDSN Event query.

  • <sp-datetime>

    A time/date chooser.

  • <sp-quake-table>

    A table of information for a list of earthquakes.

  • <sp-helicorder>

    A 24 hour helicorder display.

  • <sp-hourmin>

    An hour minute chooser.

  • <sp-latlon-box>

    An latitude/longitude box input.

  • <sp-latlon-choice>

    A component to choose a geographical area by one of several methods, including box and radius.

  • <sp-latlon-radius>

    An input for a center location and min and max radius.

  • <sp-minmax>

    A generic component to choose a range between a min and a max value.

  • <sp-organized-display>

    A container or organzied plotting of seismographs, map and info table.

  • <sp-particle-motion>

    A x-y particle motion plot from two seismograms.

  • <sp-seismogram-table>

    A table of information for seismograms.

  • <sp-seismograph>

    A plot of one or more seismograms.

  • <sp-spectra>

    A frequency domain plot of a seismogram, amplitude or phase versus frequency.

  • <sp-station-quake-map>

    A map with triangles for stations and sizable circles for earthquakes.

  • <sp-station-quake-table>

    A table of information for seismograms, stations and earthquakes.

  • <sp-station-table>

    A table of information for a list of stations.

  • <sp-timerange>

    A chooser for a range of time.
